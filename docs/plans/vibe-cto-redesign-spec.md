# VIBE CTO 리디자인 개발 명세서 (Claude Code용)

## 1. 프로젝트 개요

이 문서는 `vibe-cto.net`의 메인 랜딩 페이지를 **신뢰형 B2B 서비스 랜딩** 방향으로 재구성하기 위한 개발 명세서다.
목표는 현재 페이지의 정보 전달 중심 구조를 **전환 중심 구조**로 바꾸고, `출시 전 CTO 검증`, `48시간 리포트`, `운영/성장 연결`이라는 핵심 가치를 더 강하게 보이도록 만드는 것이다.

## 핵심 섹션 순서
1. Header
2. Hero
3. Trust Band
4. Deliverables Section
5. Sample Report Preview
6. Process Section
7. Target Customer Section
8. FAQ
9. CTA + Short Form
10. Footer

## 디자인 방향: Trust-first 엔터프라이즈형
- 화이트/라이트 그레이 베이스
- 짙은 네이비 텍스트
- 보라/블루 계열 포인트 컬러 (#6D5EF5, #4F8CFF)
- 카드 중심 UI, 신뢰 배지, 수치 강조

## 제거/후순위 대상
- Quick answers, Search summary, English summary, AEO/GEO 블록
